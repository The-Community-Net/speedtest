ကိုယ်ရေးအချက်အလက်မူဝါဒ

ဤ HTML5 အမြန်နှုန်းစစ်ဆေးသည့် ဆာဗာတွင် telemetry စနစ်ကို ဖွင့်ထားပါသည်။

ကျွန်ုပ်တို့စုဆောင်းသော အချက်အလက်များ

  • စမ်းသပ်မှု ID
  • စမ်းသပ်သည့်အချိန်
  • စမ်းသပ်မှုရလဒ်များ (ဒေါင်းလုဒ်၊ အပ်လုဒ်၊ နှောင့်နှေးမှုနှင့် အနှောင့်အယှက်)
  • IP လိပ်စာ
  • ISP အချက်အလက်
  • ခန့်မှန်းခြေ တည်နေရာ (IP လိပ်စာမှတစ်ဆင့် သိရှိခြင်းဖြစ်ပြီး GPS မဟုတ်ပါ)
  • User agent နှင့် ဘရောက်ဆာ အချက်အလက်
  • စမ်းသပ်မှု မှတ်တမ်း (ကိုယ်ရေးကိုယ်တာ အချက်အလက်များ မပါဝင်ပါ)

အချက်အလက်များကို အသုံးပြုပုံ

  • စမ်းသပ်မှုရလဒ်များကို မျှဝေနိုင်ရန်
  • ဝန်ဆောင်မှုများကို ပိုမိုကောင်းမွန်အောင် ပြုလုပ်ရန်

မည်သည့် ကိုယ်ရေးကိုယ်တာ အချက်အလက်ကိုမျှ ပြင်ပအဖွဲ့အစည်းများထံ ထုတ်ဖော်ပြောကြားခြင်း မရှိပါ။

သင်၏ သဘောတူညီချက်

စမ်းသပ်မှုကို စတင်ခြင်းဖြင့် သင်သည် ဤမူဝါဒပါ စည်းကမ်းချက်များကို သဘောတူညီပြီး ဖြစ်ပါသည်။
